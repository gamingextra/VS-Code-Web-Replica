Features

Everything You Need to Code

A complete browser-based development environment with professional-grade features you expect from a desktop IDE.

Full VS Code UI

Activity Bar, Sidebar, Editor with tabs, Terminal, Status Bar, and Command Palette — every panel faithfully recreated in the browser.

AI-Powered Coding

Kilo Code integration delivers inline completions, conversational chat, and intelligent code generation right where you need it.

5 Built-in Themes

Dark+, Light+, Monokai, Solarized Dark, and GitHub themes included out of the box. Switch instantly with one click.

Built-in Terminal

Full terminal emulator with shell commands, output streaming, and multi-tab support. Run anything you'd run locally.

Code Execution

Run code in secure Go-powered sandboxes with real-time output streaming. Support for multiple programming languages.

Microservice Architecture

TypeScript Core API, Go Sandbox, Rust Search, and AI Copilot — each service independently deployable and scalable.

Real-time Collaboration

WebSocket-powered connections with live service health monitoring. Code together with your team in real time.

Responsive Design

Works seamlessly on mobile, tablet, and desktop. Full editing experience no matter what device you're on.

Extension System

Browse and install from the built-in extensions marketplace. Customize your editor with themes, languages, and tools.

Git Integration

Full SCM view, inline diff viewer, branch management, and commit history. Version control without leaving the browser.

Cloud-Native Microservices

Built with independent, scalable services that communicate through well-defined APIs.

Quick Start

Up and Running in Minutes

Clone, install, and start coding. It's that simple.

Terminal
# Clone the repository
$git clone https://github.com/gamingextra/VS-Code-Web-Replica.git
$cd VS-Code-Web-Replica

# Install dependencies
$npm install

# Start the development server
$npm run dev

# Or start with Docker Compose
$docker compose up -d
